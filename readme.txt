Насколько смог подробно, весь процесс установки ниже расписал,
если что-то не заработает пиши на каком этапе проблемы,
если все будет нормально тоже отпишись, я по структуре и коду все расскажу

Устанавливаем:
Node.js (https://nodejs.org/en/, v6)
PostgreSQL (желательно) или MySQL

В консоли проверяем что нода и вместе с ней нпм установились нормально:
node --version
npm --version
Обе команды должны вернуть номера версий

Устанавливаем через консоль пакет для заполнения БД данными:
npm install -g sequelize-cli

В СУБД создаем новую БД

В папке приложения редактируем файл "app/config/database.js":
Там редактируем объект "development",
	username/password - данные пользователя с которого подключаемся к СУБД
	database - название БД, созданной на предыдущем шаге
	host - не меняем
	если используем Postgres, port/dialect также не меняем
	если MySQL, port - 3306 (по умолчанию), dialect - mysql

Переходим в консоли в папку приложения (LPK) и выполняем:
npm install --production

Если используем MySQL, далее выполняем:
npm install --save mysql

Запускаем приложение:
npm run start:dev

Если все прошло без ошибок должно появится сообщение о том что сервер слушает
на 8000 порте, теперь можно в браузере открыть localhost:8000/
Логин/пароль - root/root
